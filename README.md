## Oppsett

### 1. Klone prosjektet
```bash
git clone <repo-url>
```

### 2. Åpne i Android Studio
Åpne prosjektmappen i Android Studio og la Gradle synkronisere avhengigheter.

### 3. Kjør appen
Kjør på emulator eller fysisk enhet med Android 7.0+ (API 24).

## Krav
- Android Studio Hedgehog eller nyere
- Android API 24+
